## Project state

This is a freshly scaffolded Cloudflare Workers project (generated by `create-cloudflare`). It currently serves a single static asset with no application code yet:

- `public/index.html` — the entire site, served as a static asset (no build step, no framework)
- `wrangler.jsonc` — Worker configuration; `assets.directory` points at `./public`

There is no server-side Worker script yet — `wrangler.jsonc` only configures static asset serving. When adding Worker logic (routes, bindings, etc.), a `main` entry pointing at a script (e.g. `src/index.ts`) will need to be added to `wrangler.jsonc`.

## Commands

- `npm run dev` / `npm start` — run locally via `wrangler dev`
- `npm run deploy` — deploy via `wrangler deploy`
- `npx wrangler types` — regenerate TypeScript types; **run this after changing any bindings in `wrangler.jsonc`**

There is no lint, test, or build script configured yet.
