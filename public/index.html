🌏 Earthquake CLI — quick guide

What is this?

A live map of recent earthquakes (from Malaysia's meteorological feed, refreshed every 15 minutes). The dark window is a terminal: you type a short command, press Enter, and it answers.

Try it now

Click inside the terminal, type list and press Enter. The latest quakes are plotted on the map and the terminal tucks itself away so you can see them — click >_ earthquake — terminal in the bottom-left corner to bring it back.

Handy commands

help every command, with its options
list the latest earthquakes, on the map
list --mag>6 only the big ones (magnitude above 6)
search Indonesia quakes whose location mentions a word
nearby 3.14 101.69 closest quakes to a spot (that's Kuala Lumpur)
stats totals, averages, records
watch --mag>5 get an alert here when a strong quake arrives
export csv download the data as a spreadsheet file
richter 7 what a magnitude actually means

The window

Drag the top bar to move it. – hides it (the map goes full screen), □ makes it big, double-clicking the bar does the same, and ? reopens this guide.

Tip: commands that show quakes on the map hide the terminal automatically — the bottom-left chip brings it back.
